
# 🚀 Mentis

An AI-powered mental wellness companion chatbot built to explore conversational support systems using Python and Streamlit.

## 📌 About
Mentis is a prototype mental health companion chatbot developed during a 6-week AI & Cloud Technology internship with the Edunet Foundation. The project focuses on creating a supportive conversational interface designed to provide emotionally aware interactions and wellness-oriented responses.

The primary goal of the project was to apply AI and cloud concepts in a practical environment while learning how conversational systems are structured and deployed.

## ✨ Features
- Interactive chatbot interface ✅ completed
- Basic conversational flow ✅ completed
- Mental wellness-focused responses ✅ completed
- Prototype deployment/testing ✅ completed

## 🚀 Getting Started

1. git clone https://github.com/preeti9717/mentis-chatbot.git

2. cd mentis-chatbot

3. pip install -r requirements.txt

4. streamlit run app.py
